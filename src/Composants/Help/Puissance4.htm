3         Le Puissance 4

 

3.1       But du jeu

Jeu de stratégie pour deux personnes, dont le but est de placer 4 jetons de même couleur, la couleur du joueur, horizontalement, verticalement ou diagonalement dans une grille.

3.2       Commencer une partie

Pour commencer une partie de Puissance 4, suivez les instructions suivantes :

Sur la Barre de Menu, cliquez sur Fichier > Choisir jeu > Puissance 4.

Si vous avez cliqué sur Fichier > Choisir jeu > Avancé au lieu de Puissance 4, la fenêtre ci-contre apparaît. Vous pouvez choisir le jeu que vous souhaitez et modifier, quand cela est permis, certains paramètres.

 

Une fois la partie terminée, si vous souhaitez rejouer, cliquez sur Nouvelle partie, ou faites le raccourci ALT + F2, afin de créer une nouvelle partie, du même jeu auquel vous jouez, avec les mêmes joueurs.

3.3       Configuration des joueurs

Lors dun partie, ou après celle-ci, vous serez peut être amené à vouloir changer de joueur ou de pseudo, pour ce faire, il y a Joueurs qui vous le permet. Voici les explications concernant cette partie du programme :

Pseudo : Entrez dans ce champ le pseudo que vous souhaitez. (Limité à 12 caractères)

Type de joueur : Sélectionner la nature du joueur.

 Si Humain, il ny a rien dautre à faire.

Si Ordinateur, vous pouvez choisir la difficulté.

Profils : Il sagit de la liste des difficultés des IA, si Ordinateur a été choisi. Si vous choisissez personnalisé, la fenêtre sagrandit et vous permet de désigner les paramètres dont vous avez envie.

Ensuite, vous pouvez aller au joueur Suivant, ou Lancer Partie!.

 

3.4       Comment jouer ?

Une fois la nouvelle partie lancée, vous obtenez une grille comme celle qui se trouve sur lillustration, ci-dessus, dans la Surface de jeu. Selon le principe du puissance 4, vous pouvez poser des pions à partir du bas, et là où il ny pas encore de pion, sinon, au-dessus. Pour poser votre pion sur le jeu, cliquez simplement sur lemplacement où vous souhaitez le poser. Si le pion ne se marque pas sur la grille, cest que vous ne pouvez pas poser le pion à cet endroit. Si vous ne savez pas où vous pouvez poser votre pion, regarder à laide de loutil fourni par Games 4 all, dans Options > Jouables ?.

(cf. Utilisation de jouables dans 1.3 Présentation de lenvironnement.)
